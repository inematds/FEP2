Tecnicas para comprimir prompts e usar variaveis para customizacao rapida.

O Conceito

Prompt compression e condensar prompts longos em versoes menores sem perder eficacia. Menos tokens = mais rapido, mais barato, mais espaco para context.

Por que Comprimir

  • $ Custo menor por request
  • ‚ö° Latencia reduzida
  • üì¶ Mais espaco para conteudo
  • üéØ Menos ruido, mais foco

Como Pedir Compressao

"Rewrite this prompt to be 50% shorter while maintaining all functionality and nuances. Output in markdown code block."

O que Sao

Variable tags sao placeholders como {CUSTOMER_NAME}, {PRODUCT}, {TONE} que voce preenche antes de usar. Torna prompts reutilizaveis para diferentes contextos.

Sintaxe Comum

{VARIABLE}
{{variable}}
[VARIABLE]
<variable>

Exemplo de Uso

Write an email to {CUSTOMER_NAME} about {TOPIC} in a {TONE} tone.

---

CUSTOMER_NAME = Joao Silva

TOPIC = renewal of contract

TONE = professional but friendly

O Recency Bias

Modelos prestam mais atencao ao inicio e FIM do prompt. Variaveis no final recebem mais "atencao" durante processamento.

Estrutura Recomendada

[CORPO DO PROMPT - instrucoes fixas]

...

...

---

VARIABLES:

CUSTOMER_NAME = valor

TOPIC = valor

SPECIAL_INSTRUCTIONS = valor

Por que no Final

  • ‚Ä¢ Recebe mais peso durante processamento
  • ‚Ä¢ Facil de encontrar e editar
  • ‚Ä¢ Nao quebra fluxo das instrucoes
  • ‚Ä¢ Compativel com automacao (append)

O Objetivo

"Make prompts easy to customize and easy to change and evaluate." Variaveis permitem isso sem editar corpo do prompt.

Workflow de Uso

  1. 1. Copie prompt template
  2. 2. Va ate secao de variaveis
  3. 3. Atualize apenas os valores
  4. 4. Execute

Automacao

Em N8N/Make.com, variaveis podem ser preenchidas dinamicamente de outras fontes (CRM, forms, webhooks).

O Problema

AI as vezes mistura texto formatado com code blocks, criando "spillover" que quebra copy/paste.

A Solucao

"Output in unformatted markdown in a code block so I can easily copy-paste. No spillover between plain text and code blocks."

Quando Usar

  • ‚Ä¢ Gerando prompts para reusar
  • ‚Ä¢ Criando templates
  • ‚Ä¢ Exportando para outras ferramentas
  • ‚Ä¢ Sempre que precisar copiar output inteiro

Dica Extra

Adicione "I'm in a bit of a rush" no final. Faz AI pular explicacoes longas e ir direto ao output.

üîß O Exercicio

Pegue um prompt longo que voce usa e transforme-o em template com variaveis.

Passo a Passo

  1. 1. Identifique partes que mudam (nomes, topicos, tom)
  2. 2. Substitua por {VARIABLE_NAME}
  3. 3. Crie secao VARIABLES no final
  4. 4. Peca ao AI para comprimir 30-50%
  5. 5. Teste com diferentes valores

Variaveis Comuns

{CUSTOMER_NAME}
{COMPANY}
{TOPIC}
{TONE}
{LENGTH}
{CONTEXT}

Resultado

Prompt template reutilizavel que se adapta a qualquer situacao com mudanca minima. Pronto para automacao.
